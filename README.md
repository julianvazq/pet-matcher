# Dog Finder

This application allows you to find dogs up for adoption in your area.

## Tech Stack

### Frontend

- React 
- [Styled Components](https://styled-components.com/) (CSS)

### Backend

- NodeJS
- Express

## Features

- Visual branding
- Responsive design
- Responsive images
- Pagination
